## Position

* `pos` Por defecto
```CSS 
position: relative;
```

* `pos:s`
```CSS 
position: static;
```

* `pos:a`
```CSS 
position: absolute;
```

* `pos:r`
```CSS 
position: relative;
```

* `pos:f`
